# ProcureDesk

ProcureDesk is a Clean Architecture ASP.NET Core Purchase Order System.

It demonstrates how to model a real procurement domain using:
- Domain-Driven Design (DDD)
- Onion / Clean Architecture
- EF Core
- REST API

## Features

- Maintain Goods
- Maintain Suppliers
- Link Goods to Suppliers with pricing and lead time
- Create and manage Purchase Orders
- Prevent ordering goods from suppliers that donâ€™t sell them
- Track overdue purchase orders

- .NET / ASP.NET Core Web API
- EF Core
- Clean Architecture
- Structurizr C4 model
